# Air-Canvas
AirCanvas is an interactive drawing application that utilizes hand gesture recognition to enable users to switch tools, such as drawing, erasing, and changing colors, without touching the screen. It leverages MediaPipe for real-time hand tracking and integrates it with a simple, user-friendly interface for drawing and modifying sketches.

# Features
Hand Gesture Recognition: Switch tools like drawing, erasing, and color changing using hand gestures.

Interactive Canvas: Draw freely on the canvas, adjust brush sizes, and erase with gestures.

Color Palette: Select colors and change the drawing color with simple gestures.

Dynamic Tool Switching: Seamlessly switch between tools with gesture control.

Requirements
Python 3.x
MediaPipe
OpenCV
Numpy
